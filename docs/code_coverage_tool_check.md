# Code Coverage Tool Check

This check records whether line/branch/toggle Verilog coverage tools are available on this machine.

| Tool | Status | Path |
| --- | --- | --- |
| Verilator | not found | n/a |
| covered | not found | n/a |

The current project therefore reports Icarus self-checking functional coverage in docs/metrics_report.md.
For strict line/branch/toggle code coverage, install Verilator or covered and reuse the same testbench set.
